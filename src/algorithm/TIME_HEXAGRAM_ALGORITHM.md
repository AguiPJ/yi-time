# 时间→卦象映射算法

> 此文档定义时间戳到卦象的映射规则。修改此文件后需同步更新 `src/utils/timeToHexagram.js`。

## 1. 时间分解

从时间戳中提取以下分量：

| 分量 | 说明 | 范围 |
|------|------|------|
| year | 公历年 | 整数 (e.g. 2026) |
| month | 月 | 1–12 |
| day | 日 | 1–31 |
| hour | 时 | 0–23 |
| minute | 分 | 0–59 |

## 2. 八卦索引

八卦按 乾兑离震巽坎艮坤 排列，索引 0–7：

| 索引 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|------|---|---|---|---|---|---|---|---|
| 卦名 | 乾 | 兑 | 离 | 震 | 巽 | 坎 | 艮 | 坤 |
| 符号 | ☰ | ☱ | ☲ | ☳ | ☴ | ☵ | ☶ | ☷ |
| 五行 | 金 | 金 | 火 | 木 | 木 | 水 | 土 | 土 |

## 3. 上卦计算（上卦 / 外卦）

```
上卦 index = (year + month + day) % 8
```

> 上卦由年月日决定，代表"天时"。

## 4. 下卦计算（下卦 / 内卦）

```
下卦 index = (hour + minute) % 8
```

> 下卦由时分决定，代表"人事"。

## 5. 动爻计算

```
动爻 index = (year + month + day + hour + minute) % 6
```

> 动爻决定卦中哪一爻在变动。索引 0–5 对应自下而上第 1–6 爻。
>
> - 动爻为阳 → 变阴（老阳）
> - 动爻为阴 → 变阳（老阴）

## 6. 本卦（原卦）

本卦由上卦和下卦组合而成：
- 上卦 3 爻在上（第 4, 5, 6 爻）
- 下卦 3 爻在下（第 1, 2, 3 爻）
- 共 6 爻，编号自下而上

## 7. 变卦

将本卦的动爻（1 爻）翻转（阳↔阴），其余 5 爻不变，得到变卦。

## 8. 互卦

互卦从本卦的中间 4 爻提取：
- 互卦下卦：本卦第 2, 3, 4 爻
- 互卦上卦：本卦第 3, 4, 5 爻

## 9. 示例演算

**时间：2026-06-07 14:30**

```
year=2026, month=6, day=7, hour=14, minute=30

上卦 = (2026 + 6 + 7) % 8 = 2039 % 8 = 7 → 坤 ☷
下卦 = (14 + 30) % 8 = 44 % 8 = 4 → 巽 ☴
动爻 = (2026 + 6 + 7 + 14 + 30) % 6 = 2083 % 6 = 1 → 第2爻

本卦 = 坤上巽下 = 升 (䷭)
  lines: [yin, yang, yang,  yin, yin, yin]
         巽下             坤上

动爻 index=1 (第2爻) 为 yang → 变 yin
变卦 lines: [yin, yin, yang,  yin, yin, yin]
           = 坤上艮下 = 谦 (䷎)

互卦 lines[1..4] = [yang, yang, yin, yin]
  互下 = [yang, yang, yin] = 兑 ☱
  互上 = [yang, yin, yin] = 震 ☳
  互卦 = 震上兑下 = 归妹 (䷵)
```

## 10. 可扩展项

以下字段预留，可通过修改算法扩展：

- **世应**：世爻和应爻位置
- **六亲**：父母、兄弟、妻财、官鬼、子孙
- **六兽**：青龙、朱雀、勾陈、螣蛇、白虎、玄武
- **纳甲**：天干地支纳甲
- **卦气**：十二月卦气
- **五行生克**：体用生克关系
